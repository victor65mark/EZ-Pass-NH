# EZ Pass NH

## Description
EZ Pass NH is a modern and efficient web-based application that simplifies toll management and account tracking for New Hampshire’s E-ZPass users. It allows drivers to monitor toll activity, add or remove vehicles, recharge balances, and review payment history in real-time. Designed with accessibility and responsiveness in mind, the app provides seamless user experience across devices. The goal is to make toll management intuitive, transparent, and secure while integrating real-time updates and receipt management for every trip. With this tool, users can efficiently manage their toll transactions and stay informed about their account activities anytime, anywhere.

---

## Features
- Secure login and user authentication  
- Real-time toll balance and transaction tracking  
- Vehicle management and account updates  
- Downloadable toll receipts  
- Responsive UI for all devices  
- Integrated expense tracking dashboard
